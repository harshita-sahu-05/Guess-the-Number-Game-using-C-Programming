# Guess the Number Game 🎯

## Overview
Guess the Number Game is a console-based application developed in **C Programming**. The game generates a random number between **1 and 100**, and the player attempts to guess the correct number. Based on the user's input, the program provides hints indicating whether the guess should be higher or lower until the correct number is found.

This project demonstrates fundamental programming concepts such as loops, conditional statements, random number generation, and user interaction.

---

## Features
- Generates a random number between **1 and 100**
- Provides hints:
  - **Higher Number Please!**
  - **Lower Number Please!**
- Tracks the number of attempts
- Displays a congratulatory message when the correct number is guessed
- Interactive command-line interface

## How It Works
1. The program generates a random number between **1 and 100**.
2. The user enters a guess.
3. The program compares the guessed number with the generated number.
4. If the guess is incorrect, the program displays a hint:
   - Higher Number Please!
   - Lower Number Please!
5. The process continues until the correct number is guessed.
6. The total number of attempts is displayed.
